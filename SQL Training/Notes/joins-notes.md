# JOINS

JOIN is a keyword in SQL used to combined matched rows from two or more tables.

They are powerful because they allow you to create a list of combined rows of matching data.

## JOIN Types

There are different JOIN types in SQL, and it is important to understand the differences between them in order to know which one to use to complete a task.

**INNER JOIN (simple JOIN)**

- Most used/most common
- Return any rows where there is a matched key in BOTH tables
